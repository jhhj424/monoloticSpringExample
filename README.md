
## 독자 (Reader), 작가(Writer) 가 존재하는 웹소설 플랫폼 구축

- 작가는 웹소설을 편당 금액과 함께 등록할 수 있음
- 독자는 돈을 지불하여 이를 감상
- 기본적인 도서에 대한 관리 필요
- 결제를 위한 내용이 필요하다.

 *추후에 캐시 적용*

- **독자는 소설을 전부 즉시 결제로만 이용 가능하다.**


### 웹소설
  - 웹 소설 1화,2화(챕터)


### 작성해야 할 것 (요구사항)

**작가**
 - [ ] 작가 등록
 - [ ] 웹 소설 등록
 - [ ] 웹 소설의 한 챕터를 등록한다 ( 금액과 함께 )
 - [ ] 작가님의 출금 ( 결제한 것들에 대한 )
 - [ ] 웹 소설 C,R,U,D , 챕터에 대한 C,R,U,D

**독자**
 - [ ] 회원가입
 - [ ] 웹 소설의 챕터를 결제한다.
 - [ ] 웹 소설의 목록을 볼 수 있다.
 - [ ] 내가 특정 소설의 결제한 목록을 볼 수 있다.
